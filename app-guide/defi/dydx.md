# dYdX

{% hint style="warning" %}
**Migration in Progress**

dYdX governance features are being migrated to the DeFi tab. Currently, these functions are still accessible via the **Functions** button on the dYdX chain in the Wallet tab.
{% endhint %}

Vultisig supports dYdX governance participation. Users can vote on dYdX proposals directly from the DeFi tab.

***

## Available Actions

| Action | Description |
|--------|-------------|
| Vote | Vote on governance proposals |
| Custom | Advanced memo transactions |

***

## Voting

### How to Vote

1. Open **DeFi tab** → **dYdX** → **Vote**
2. Enter the **Proposal ID**
3. Select your **vote** (Yes/No/Abstain)
4. Sign with your devices

{% hint style="info" %}
To participate in governance, you must hold DYDX tokens in your vault.
{% endhint %}

***

## Custom Transactions

Advanced users can create custom memo transactions to interact with the dYdX ecosystem.

{% hint style="warning" %}
Custom transactions are for advanced users only.
{% endhint %}
